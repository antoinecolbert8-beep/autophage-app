## 🚀 **LANCEMENT EN 48H - CHECKLIST**

### **Jour 1 : Configuration (4-6h)**

#### **1. Stripe (1h)**
```
https://dashboard.stripe.com/products
```
- Créez 3 produits (Starter, Pro, Business)
- Configurez le webhook `/api/webhooks/stripe`
- Copiez les Price IDs dans `.env`

#### **2. Base de données (30min)**
```bash
# Remplacer le schéma
cp prisma/schema-saas.prisma prisma/schema.prisma

# Migrer
npx prisma db push
```

#### **3. Vertex AI (1h)**
```
https://console.cloud.google.com/billing
```
- Activer facturation
- Activer Vertex AI API
- Activer Text-to-Speech API

#### **4. Déploiement Vercel (1h)**
```bash
vercel --prod
```
- Configurez toutes les variables d'environnement
- Connectez votre domaine

#### **5. DNS (30min)**
```
A record : @ -> Vercel IP
CNAME : www -> cname.vercel-dns.com
```

### **Jour 2 : Marketing & Lancement (4-6h)**

#### **6. Contenu (2h)**
- Filmez 3 vidéos de démo
- Créez 10 posts LinkedIn
- Préparez les screenshots

#### **7. SEO (1h)**
- Créez 5 articles de blog
- Optimisez les meta tags
- Soumettez à Google

#### **8. Product Hunt (1h)**
```
https://www.producthunt.com/posts/new
```
- Préparez le lancement
- Invitez 20 friends à upvoter
- Planifiez pour un mardi

#### **9. Go Live (2h)**
- Annoncez sur LinkedIn
- Envoyez aux early adopters
- Activez les pubs Google
