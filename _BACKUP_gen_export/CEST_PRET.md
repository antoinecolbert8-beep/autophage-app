## 🔑 AJOUTER D'AUTRES CLÉS API (Optionnel)

Si vous voulez activer plus de fonctionnalités, éditez le fichier `.env.local` :

### Pour la Téléphonie (Twilio)
```env
TWILIO_ACCOUNT_SID="votre_sid"
TWILIO_AUTH_TOKEN="votre_token"
TWILIO_PHONE_NUMBER="+33123456789"
```

### Pour OpenAI (GPT-4)
```env
OPENAI_API_KEY="sk-..."
```

### Pour WhatsApp Business
```env
WHATSAPP_PHONE_NUMBER_ID="123456789"
WHATSAPP_ACCESS_TOKEN="EAAxxxxxxx"
```

### Pour Instagram/Facebook
```env
META_PAGE_ACCESS_TOKEN="EAAxxxxxxx"
META_PAGE_ID="123456789"
META_INSTAGRAM_ACCOUNT_ID="123456789"
```

### Pour TikTok
```env
TIKTOK_ACCESS_TOKEN="act.xxxxxxx"
```

### Pour YouTube
```env
YOUTUBE_API_KEY="AIzaSy..."
YOUTUBE_CLIENT_ID="xxx.apps.googleusercontent.com"
YOUTUBE_CLIENT_SECRET="xxx"
```

## 🛠️ COMMANDES UTILES

```bash
npm run dev
```

### Arrêter le Serveur

### Vérifier la Base de Données
```bash

```

### Lancer le Bot LinkedIn
```bash
cd SaaS_Bot_LinkedIn
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
playwright install chromium
python login_saver.py
```

```bash
docker-compose up -d
```
